# AI Resume Analyzer 🚀

AI Resume Analyzer is a Flask-based web application that helps users evaluate resumes using ATS (Applicant Tracking System) scoring logic.

## Features

* Upload Resume in PDF or TXT format
* Paste Resume Text directly
* ATS Score Calculation
* Skill Detection
* Strong Action Verb Analysis
* Quantified Achievement Detection
* Resume Improvement Suggestions
* Modern Futuristic UI

## Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* PDFPlumber
* Gunicorn
* Render Deployment

## Project Description

Built an AI-powered Resume Analyzer using Flask, PDF parsing, and ATS scoring logic to evaluate resumes, detect technical skills, analyze impact metrics, and provide improvement suggestions for better job opportunities.
